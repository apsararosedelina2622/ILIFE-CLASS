==> Cloning Files from github to local

==> Commands

    ~ git clone "link"
    ~ cd demo 
    ~ git add filename (git add index.txt, git add .)
    ~ git commit -m 'commit'
    ~ git push
    ~ git pull 

===> Cloning files from local to github

    Create Folder and Files
    Go to Source Control -> Then Commit -> Publish Branch -> Set Repository Name -> Select Public or Private 

===> Hosting an website using github

    Open your repository -> Settings -> Pages -> Branch (main , /root) -> Click Save 
    After few seconds, hosted link will be shown
